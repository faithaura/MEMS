# UzimaNode Emergency Medical Response System

## Overview
This repository contains the code for the UzimaNode emergency medical response system, which is implemented using React, Node.js, Tailwind CSS, and MongoDB, all within a Vite-based setup.

## Project Structure

### Frontend (client/)

- **client/package.json**: Configuration for the React application.
- **client/vite.config.js**: Vite configuration file.
- **client/tailwind.config.js**: Tailwind CSS configuration.
- **client/postcss.config.js**: Config for PostCSS.
- **client/index.html**: Main HTML file.
- **client/src/**: Directory containing source code.

### Backend (server/)

- **server/package.json**: Configuration for the Node.js application.
- **server/.env.example**: Example environment variables for the server.
- **server/src/**: Main source directory for the server.

### Shared/Documentation

- **docs/**: Directory containing documentation files.

## Getting Started
Follow the setup instructions in the docs directory to get started with running this project locally.
